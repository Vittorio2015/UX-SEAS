Create Stack of Cloud Resources

Select Instance Specifications:

* Must contain only letters, numbers, dashes and start with an alpha character.

* Choose a user name to login to the new server.

* Type a SSH public key that will be installed on the user home directory on the new server.

* Type comma separated port numbers, which the new server will open exclusively to the Source Network.

* Type the ports that are exclusively open to this network (use 0.0.0.0/0 to access the server from the internet)
